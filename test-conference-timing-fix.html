🔧 Test Conference Timing Fix

🚨 Vấn đề đã phát hiện:

Timing Issue: useConferencePermissions được gọi với {isAuthenticated: false, user: null} mặc dù user đã được authenticate.

Nguyên nhân: Hook được gọi trước khi useAuth hoàn thành việc load user data.

Kết quả: conferencePermissions bị set về [] và không có quyền truy cập.

✅ Các thay đổi đã thực hiện:

  • Thêm authLoading check: Kiểm tra authLoading trước khi gọi API
  • Wait for auth: Chờ auth hoàn thành loading trước khi fetch permissions
  • Prevent premature redirects: Return false trong permission checks khi đang loading
  • Update isLoading: Bao gồm authLoading trong isLoading state

🔧 Technical Details

Vấn đề đã sửa:

  • Timing issue giữa useAuth và useConferencePermissions
  • Gọi API trước khi auth hoàn thành loading
  • Permission checks trả về false khi đang loading

Giải pháp:

  • Thêm authLoading check trong useEffect
  • Chờ auth hoàn thành trước khi fetch permissions
  • Return false trong permission checks khi đang loading
  • Bao gồm authLoading trong isLoading state
